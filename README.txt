PL::

Założenia projektu:
Gra tamagochi, przynajmniej 5 różnych parametrów o które należy dbać. Stan powinien być zapisywany i odczytywany z pliku przy każdym uruchomieniu. 

Projekt Tamagochi powstał podczas kursu KKZ do zawodu technik programista.

Projekt został napisany w języku C++ obiektowym aby go uruchomić zalecany jest VisualStudio Community 2019 i nowszy

ENG::

Project assumptions:
Tamagochi game, at least 5 different parameters to be taken care of. The state should be written to and read from a file every time it is run. 

The Tamagochi project was created during the KKZ course for Programming Technician.

The project was written in C++ object-oriented language to run it VisualStudio Community 2019 and newer is recommended
